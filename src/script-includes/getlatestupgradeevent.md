API Name: global.getLatestUpgradeEvent

```js
function getLatestUpgradeEvent() {
	var gr = new GlideRecord('sys_performance_event');
	gr.addQuery('event_type', 'upgrade');
	gr.orderByDesc('sys_created_on');
	gr.query();
	if (gr.next())
		return gr.sys_id;
	
	return '';
}
```
